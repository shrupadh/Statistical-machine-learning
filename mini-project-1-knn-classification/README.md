# Mini Project 1: K-Nearest Neighbors (KNN) Classification

## Project Overview

This project implements the **K-Nearest Neighbors (KNN)** classification algorithm for supervised classification using both **Python** and **R**. The objective is to evaluate model performance across multiple values of **K**, identify the optimal classifier, and compare implementations between programming languages.

This project was completed as part of the **Statistical Machine Learning** course at the **University of Texas at Dallas**.

---

## Objectives

- Implement the K-Nearest Neighbors (KNN) classification algorithm.
- Train and evaluate classification models using multiple K values.
- Identify the optimal K based on prediction accuracy.
- Compare KNN implementations in Python and R.
- Visualize classification performance using model evaluation techniques.

## Machine Learning Workflow

1. Load the training and testing datasets.
2. Explore and preprocess the data.
3. Train KNN models using multiple values of K.
4. Predict class labels for the testing dataset.
5. Evaluate model accuracy.
6. Select the optimal K value.
7. Compare Python and R implementations.
8. Summarize model performance.
